# Home Lab de Cybersecurity – Laboratório Inicial

Este repositório registra a construção do meu primeiro laboratório de Segurança da Informação, usando Kali Linux em máquina virtual.  
O objetivo é ter um ambiente controlado para estudar conceitos básicos de redes, portas, serviços e varredura com Nmap, sem risco para o sistema operacional real.

---

## Ambiente utilizado

- Host: Windows com VirtualBox
- Máquina virtual: Kali Linux
- Modo de rede: NAT
- Usuário no Kali: pedrotech

---

## O que já foi feito

- Instalação do VirtualBox no Windows  
- Download da ISO oficial do Kali Linux  
- Criação da máquina virtual  
- Instalação do Kali Linux (Graphical Install)  
- Primeiro login com o usuário pedrotech  
- Atualização inicial do sistema com apt  
- Identificação do endereço IP da máquina com ip a  
- Primeira varredura de portas usando sudo nmap <ip-da-máquina>  
- Registro de prints e anotações técnicas

O relatório técnico da varredura inicial está em:

relatorios/relatorio-nmap-inicial.md
